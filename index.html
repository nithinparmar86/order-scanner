One-time Setup

Uses Firebase (free, by Google) to sync your order and scan counts instantly across every device.

1
Go to console.firebase.google.com → Add project → name it → Create
2
Left sidebar → Build → Realtime Database → Create database → Test mode → Enable. Copy the URL shown.
3
Gear ⚙ → Project settings → General → Add app → Web → register → copy apiKey and projectId.
Share these 3 values with staff — they enter them once and stay connected.
